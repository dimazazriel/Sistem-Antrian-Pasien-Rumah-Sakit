# Sistem-Antrian-Pasien-Rumah-Sakit
Sistem untuk menangani antrian pasien dengan prioritas berdasarkan tingkat keparahan,
menggunakan priority queue untuk pengelolaan data pasien.

## Deskripsi:
Implementasi Struktur Data untuk Pemecahan Masalah Nyata Tujuan

## Proyek:
Proyek ini bertujuan untuk meningkatkan pemahaman mahasiswa tentang konsep dasar dan
penerapan struktur data dalam menyelesaikan permasalahan nyata. Mahasiswa akan bekerja
dalam kelompok untuk mengembangkan program berbasis teks menggunakan bahasa C.

## Deskripsi Tugas:
Mahasiswa diminta untuk memilih salah satu dari alternatif topik yang disediakan atau
membuat judul sendiri. Setiap kelompok akan mengembangkan aplikasi yang relevan dengan
topik tersebut, memanfaatkan struktur data seperti array, linked list, stack, queue, tree, atau
graph. Proyek harus mencakup fungsi dasar seperti penambahan data, penghapusan, pencarian,
dan menampilkan data.

## Rincian Pengerjaan Tugas
* Struktur Data:
Mahasiswa wajib menggunakan struktur data yang sesuai dengan kebutuhan proyek.
Setiap fungsi harus diimplementasikan dengan efisien dan menggunakan konsep
struktur data yang tepat.
* Fitur Wajib:
  1. Menambah data.
  2. Menghapus data.
  3. Mencari data.
  4. Menampilkan data secara terstruktur.
   
## Dokumentasi:
1. Jelaskan topik yang dipilih, struktur data yang digunakan, dan alasan memilihnya.
2. Sertakan penjelasan bagian bagian kode program yang dibuat.
   
## Kriteria Penilaian:
1. Pemahaman Konsep (20%)
   - Struktur data yang dipilih sesuai dengan masalah.
   - Penjelasan fungsi dalam kode.
2. Implementasi dan Kualitas Kode (40%) 
   - Program berjalan sesuai spesifikasi.
   - Kode rapi dan sesuai standar pemrograman.
3. Fungsionalitas Aplikasi (30%)
   - Aplikasi memiliki fitur lengkap sesuai deskripsi topik.
   - Dapat menangani berbagai masukan pengguna, termasuk masukan tidak valid.
4. Dokumentasi dan Presentasi (10%)
   - Kelengkapan dokumentasi:
        1) Makalah Lengkap
        2) PPT
        3) Luaran dalam format jurnal (ringkasan)
   - Kemampuan menjelaskan proyek dalam presentasi.

## Batasan dan Catatan Penting:
* Kelompok: Masing-masing kelompok terdiri dari 3-4 orang.
* Durasi Proyek: 4 minggu.
* Bahasa Pemrograman: C, berbasis teks.
* Originalitas: Plagiarisme tidak diperbolehkan; setiap proyek harus dibuat dari nol dengan kreativitas mahasiswa.
